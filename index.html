CH 01

認識程式設計

1-1 什麼是程式？

程式（Program）是由一連串指令所組成，用來告訴電腦如何執行特定的工作。

名詞說明比喻
演算法（Algorithm）解決問題的步驟與方法食譜
程式（Program）用程式語言撰寫的演算法依照食譜寫下的步驟
軟體（Software）由一個或多個程式組成的系統整本食譜書
💡
程式設計師（Programmer） 負責撰寫程式，讓使用者（User）能透過程式完成任務，例如：計算成績、傳送訊息、播放音樂。

1-2 程式語言種類

類型說明範例
機器語言由 0 和 1 組成，電腦直接執行10110000 01100001
高階語言接近人類語言，需翻譯才能執行Python、C/C++、Java

Python 的特色：語法簡潔、功能強大（AI/網頁/資料分析）、跨平台、免費開源。

CH 07

while 迴圈

7-1 while 基本語法

當「不確定要重複幾次」時使用 while 迴圈，條件成立就繼續執行。

⚠️
一定要確保條件最終會變成 False！否則會造成無窮迴圈，程式永遠不停止。
python
i   = 1
sum = 0
while i <= 5:
    sum += i
    i   += 1    # ← 非常重要！讓 i 增加
print('總和：', sum)  # 15

7-2 break 與 continue

python
# break：完全跳出迴圈
for i in range(1, 6):
    if i == 3:
        break
    print(i, end=' ')  # 1 2

# continue：跳過本次，繼續下一次
for i in range(1, 6):
    if i == 3:
        continue
    print(i, end=' ')  # 1 2 4 5

7-3 綜合範例：猜數字遊戲

python
k     = 16   # 答案
guess = 0
t     = 0    # 猜測次數
print('=== 猜數字遊戲（1-20）===')
while guess != k:
    t     += 1
    guess = int(input('猜測：'))
    if   guess > k:  print('猜大了！')
    elif guess < k:  print('猜小了！')
    else:           print('答對了！共猜', t, '次')
=== 猜數字遊戲（1-20）=== 猜測：10 猜小了！ 猜測：18 猜大了！ 猜測：16 答對了！共猜 3 次

🔥 困難 練習 7-2｜成績輸入系統 ▼
設計成績輸入系統：輸入學生姓名後，重複輸入成績（輸入 -1 結束），最後輸出總和、平均（小數一位）、最高分。
python
name    = input('學生姓名：')
total   = 0;  count = 0;  highest = 0
score   = int(input('成績（-1結束）：'))
while score != -1:
    total   += score
    count   += 1
    if score > highest:  highest = score
    score   = int(input('成績（-1結束）：'))
print(name, '總和：', total)
print(name, '平均：', round(total/count,1))
print(name, '最高分：', highest)
